## Configuração inicial

### Configuração .env

Copie o `.env.example` e renomeie para `.env`. Preencha os dados do email de envio e seu banco de dados.

### Instalar dependencias

Para instalar as dependencias rode:

```bash
npm install
```

ou

```bash
yarn install
```

### Rodar projeto

```bash
npm start
```

ou

```bash
yarn start
```
